# 学生签到系统

## 项目简介
1.学生签到系统是基于Java Swing、Mysql数据库的签到系统。
2.角色分为：管理员、老师、学生
3.功能上：签到、签到统计、学生管理、教师管理、课程管理
4.系统包含七张数据库表

Java Swing 、Mysql

## 开发工具
Eclispe 或者MyEclipse 或者IntelliJ IDEA

## 项目资料包含
1. 前后端源代码
2. 开题报告（任务书）+ 配套论文（一万两千字以上）+ 答辩PPT
3. 远程控屏调试（向日葵或者toDesk控屏）
4. 一键启动项目（双击运行演示，无需搭建复杂环境）
5. 项目讲解视频教程
